Callculates and then applies the effect of buoyancy on a PhysicsData object

This method uses a complex calculation based on the hull geometry of the data object. In general it will yield more realistic effects but operates on each vertex in the mesh. For complex meshs this can take a long time.

static Vector3 HeathenEngineering.PhysKit.PhysKitMaths.FastIntercept ( Vector3  position,
float  speed,
Rigidbody  targetSubject 
)
static

Find the point of intercept;

This is a faster calculation than SafeIntercept but does not handle situations where there is no solution such as a target moving away faster than the interceptor moves toward

Fast intercept is appropreate for projectile to target such as a bullet to a character or vehicle where the bullet is substantially faster than the subject however for other intercept cases where the interceptor is nearer the speed of the interceptee this method may result in frequent misses in such cases use the SafeIntercept and test for null result indicating no solution

Parameters
positionthe point in space to start from
speedthe speed of travel
targetSubjectthe subject to target
Returns

◆ RelativeHeading()

static Vector3 HeathenEngineering.PhysKit.PhysKitMaths.RelativeHeading ( Quaternion  subjectRotation,
Vector3  relativeDirection 
)
static

Returns the heading vector (direction vector) for a quaternion rotaiton e.g.

to get the 'Forward' heading from a rotation use Vector3.Forward for the relative direction

Use case ... to have an object 'step to its right' you need to move it along the relative Vector3.right e.g. the right heading In cases where you only have the object's quaternion rotaiton you can use this method to find that heading such that object.position + RelativeHeading(object.rotation, Vector3.right) * distance; will move the object to its right and measure of 'distance'

Parameters
rotationThe rotaiton of the subject
relativeDirectionThe relative direction e.g. forward = 0,0,1, up = 0,1,0, etc.
Returns
