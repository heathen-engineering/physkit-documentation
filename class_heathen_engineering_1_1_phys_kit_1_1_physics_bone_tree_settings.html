Detailed Description

Defines a set of PhysicsBoneTree settings.

This allows a developer to save configurations for use on multiple trees accross multiple projects.

◆ drag

float HeathenEngineering.PhysKit.PhysicsBoneTreeSettings.drag = 0

Drag property, this discribes the strength of the effect of drag on the forces applied

0 is none, 1 is 100% and thus the node wont move. Drag serves to slow or 'dampen' movement in all directions e.g. to and from the position of rest.
